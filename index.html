Proyectos Realizados

Prototipo Gestor de Horarios

Proyecto realiazado en Angular Front-end y Java Back-End
El proyecto consisite en la implementacion de horarioa por salones para la universidad, para ello se implemento un CRUD para el manejo de recursos en la universidad, tambien se implementa la asignacion de estos recursos. Por otra parte se incorporo una funcion que permite a los ususarios agregar Un horario a un recurso especifico, selecionando Dia, Hora Inicio, Hora Fin y la materia que se le desea asiganar a ese recurso o salon de clases ademas se hizo las respectivas validadciones en las cuales destacan que no se permite asignar una clase con horas inconsistentes al igual de que no pueden matricular si el salon esta ocupado en esa franja, y demas validaciones tecninas para el llenado de datos
